## Цель проекта

Цель проекта является практика навыков разработки веб-приложений с использованием новых библиотек которые давно хотел изучить.  

В ходе работы внимание было уделено:
- работе со стилями с использованием Bootstrap
- реализации drag & drop функциональности с помощью dnd-kit
- изучению и применению localStorage для хранения данных на стороне клиента
- использованию html2canvas для создания снимков интерфейса

Проект также направлен на улучшение понимания архитектуры React-приложений и взаимодействия пользовательского интерфейса с данными.

## Как запустить проект?
```bash

# Скопировать проект
git clone https://github.com/HELLRAID-cmd/StageTask.git

# Перейти в папку react
cd react

# Установить зависимости
npm i

# Запустить проект
npm run dev
```
