## Цель проекта:
- Практика работы с React, TypeScript, dnd-kit и библиотеков Bootstrap, а также создание небольшого проекта для портфолио. Сайт сохраняет все ваши проекты и задачи в LocalStorage, это нужно чтобы был смысл существования этого проекта.

## Как запустить проект?
```bash

# Скопировать проект
git clone https://github.com/HELLRAID-cmd/StageTask.git

# Перейти в папку react
cd react

# Установить зависимости
npm i

# Запустить проект
npm run dev
```

## Адаптивность:
Проект не оптимизирован под мобильные устройства.
Минимальная комфортная ширина просмотра — 750px.
